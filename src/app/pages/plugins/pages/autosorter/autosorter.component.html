Автосортировка

Для реализации Автосортировки используются 'Раздатчики'.

Нужно разместить раздатчик, на нем рамку, в которую вставить око эндера. Далее для всех предметов, попадающих в этот раздатчик, будет происходить поиск места отправки и сама отправка предметов.

Местом отправки может быть:

  • контейнер с рамкой сверху, содержащей предмет для отправки;
  • контейнер с рамкой сверху, содержащей шалкер, внутри которого есть предметы для отправки;
  • котел с лавой внутри и рамкой сверху (в этом случае предмет будет уничтожен);
  • контейнер с рамкой сверху, содержащей удочку с морковкой (служит как шаблонный предмет, чтобы отправлять туда все предметы, которые не нашли себе место отправки из списка выше).

Рамки для мест отправки должны быть непрозрачными.

Помимо раздатчика можно использовать комбинацию из блоков: снизу 'Позолоченный чернокамень (чернит)', над ним 'Плачущий обсидиан'. В этом случае будет работать как воронка, предметы, брошенные на блок плачущего обсидиана сверху, будут попадать в систему.
